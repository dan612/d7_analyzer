## How to use
Add this to your D7 project composer.json file in the repositories section:
```
"dan612/d7_analyzer": {
    "type": "vcs",
    "url": "https://github.com/dan612/d7_analyzer.git"
}
```
Then run:
```
composer require dan612/d7_analyzer
```
Once the package is included, run the setup:
```
// some command
```
